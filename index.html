I like working on odd projects, here is a small sample of them.

Git Art

This is an Golang Application that allows you to input designs or a json and a year and be given a repository that can create art in your github history. I recommend using a year where you have no existing history, because github seems to be quicker to pick up those changes.
Color Randomizer

This is a RESTful API that allows people to get random colors displayed. It was mostly a fun exercise in creating a simple RESTful API in multiple languages to prove out how they can be easily swapped out for a UI once built. Current implementations are Node.js, Python, & Ruby is in progress.
